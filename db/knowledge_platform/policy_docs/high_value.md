# High-Value Refund Approval

Policy owner: Finance Controls. Version 2025.1. Effective 2025-01-01.

Refunds above USD 1,000 or the local-currency equivalent require manual approval by Finance Operations. Splitting one transaction into multiple smaller refunds to avoid the threshold is prohibited. The threshold applies to the total expected refund, including tax, shipping, and prior partial refunds associated with the order.

An agent may collect evidence and recommend an outcome, but it must create no payment instruction for a high-value case. Required evidence includes order identity, customer identity, fulfillment state, tender type, prior-refund history, and the current policy version. Missing amount or currency data is itself a reason to escalate.
